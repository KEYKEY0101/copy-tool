# 資料夾批量複製工具

適用於從雲端同步資料夾（如 Box Drive）批量複製大量資料夾/檔案到本機或公共盤。

## 功能
- GUI 介面，可瀏覽選擇來源和目標資料夾
- 進度條和即時日誌顯示
- 自動跳過已複製的項目，中斷後重開會接著繼續
- 失敗自動記錄

## 使用方法
1. 確保電腦已安裝 [Python](https://www.python.org/downloads/)（安裝時勾選 Add Python to PATH）
2. 雙擊 `run.bat` 啟動
3. 點「瀏覽」選擇來源和目標資料夾
4. 點「開始複製」

## 需求
- Windows
- Python 3.6+（只用內建套件 tkinter，不需額外安裝）
